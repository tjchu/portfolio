OCAML & Scala

October - December 2015

Learning two new langauges over span of three months

UCSD gave me the chance to have an intense three-months learning of two programming languages called OCAML and Scala. Both languages were heavily object-oriented and gave me huge insights into the different style and features of programming languages.

The first langauge I learned was OCAML. It is a ML-derived language and is best known for its static type systems and type-infering compilers.

Second lanague Scala that I learned was also very interesting. It's similar to scripting languages where the syntax is very concise. Variable types are usually inferred in the code by the compilers.

If you would like to see the programming assignments, feel free to view the code on my Github page.

OCAML

I created numerous functions in OCAML; this gave the chance to gain insights into how this ML-style language works. For example, I created a function sumList in one of the OCAML files that is able to find the sum of all the elements in a list variable. Another example is the digitsOfInt function. This function is able to append all the single digit elements in the list into an actual number with all the digits.

I also learned many other concepts such as tail recursion, image pixel manipulation, and implementing library functions.

Scala

I used the language Scala to create many different programs. For example, I created a program that is able parse a string with a particular rule of grammar. The program could process various mathematcial equations with symbols like +, -, *, /, =, != and many others. I also used the language to create a program that could process a text file and find the frequency of a specific word found.
